Simple Loan Portfolio Analysis (Python)

This project demonstrates basic portfolio-level analysis using a simplified loan dataset.

Objective

To reinforce core private credit and portfolio concepts using foundational Python tools.

Dataset

  • Outstanding exposure
  • Credit rating
  • Loan margin

Analysis Performed

  • Total portfolio exposure
  • Weighted average margin (exposure-weighted)
  • Exposure by credit rating
  • Simple loss sensitivity

Tools Used

  • Python
  • pandas
  • numpy
